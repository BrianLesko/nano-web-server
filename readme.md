
# Arduino Nano esp32 Web receiver
This code implements Web communication (UDP) on an arduino nano esp 32 written in Micropython. The protocol used here is called UDP and is much faster than TCP. Ive used it to control stepper motors in real time with a sony Dualsense PS5 controller.


&nbsp;

&nbsp;

## Dependencies

This code uses the following libraries:
- `socket`: for making a network interface and receiving data
- `machine`: The arduino nano uses the machine library to access its pins
- `network`: for connecting to a wifi network
- `time`: for timeout capabilities


&nbsp;

## Usage

You'll need to install micropython on an arduino nano esp32 to run this code, then swap out your wifi name and password within server.py next, you can open a socket on your laptop or other device to send data to your arduino.
